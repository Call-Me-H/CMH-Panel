# 📖 معرفی

پنل CMH یک پنل مدیریت پروکسی سبک است که کاملاً بر بستر **Cloudflare Workers** و **Cloudflare D1** ساخته شده است.

به جای اضافه کردن بی‌نهایت تنظیمات پیچیده، پنل CMH دقیقاً روی امکاناتی تمرکز دارد که کاربران واقعاً به آن‌ها نیاز دارند.

اهداف اصلی این پروژه عبارتند از:

- ⚡ نصب و راه‌اندازی سریع
- 🔒 اتصالات پایدار
- 📦 تولید لینک‌های سابسکریپشن تمیز و استاندارد
- 🚀 سازگاری بالا با کلاینت‌های محبوب
- 🛡️ فرگمنت (Fragment) قابل اعتماد برای دور زدن سیستم‌های فیلترینگ (DPI)

---

# ✨ ویژگی‌های کلیدی

## 🛡️ هسته پروکسی و اتصالات

- پشتیبانی از VLESS و Trojan
- مسیریابی در لبه کلودفلر (Edge Routing) از طریق WebSocket
- پشتیبانی از حالت‌های TLS و Non-TLS
- پشتیبانی از پورت‌های مختلف
- امکان شخصی‌سازی فینگرپرینت کلاینت
- فرگمنت هوشمند (شخصی‌سازی طول، فاصله زمانی و پکت‌ها)

---

## 👥 مدیریت کاربران و ترافیک

- محدودیت ترافیک کل و روزانه (بر اساس گیگابایت / درخواست‌ها)
- مدیریت تاریخ انقضای کاربران
- نمایش آمار لحظه‌ای کلودفلر (Cloudflare Analytics)
- فعال‌سازی، غیرفعال‌سازی و ریست ترافیک تنها با یک کلیک

---

## 🤖 مدیریت از طریق ربات تلگرام

کل پنل را مستقیماً و بدون نیاز به ورود به سایت، از طریق تلگرام مدیریت کنید.

- ساخت، ویرایش و حذف کاربران
- نظارت بر مصرف ترافیک
- مشاهده آمار مصرف کلودفلر
- روشن یا خاموش کردن کل سیستم
- بدون نیاز به ورود به دشبورد وب

## 🔗 سیستم سابسکریپشن (اشتراک)

- سابسکریپشن‌های Base64
- خروجی استاندارد Clash YAML

- صفحه اختصاصی و زیبا برای هر سابسکریپشن
- نمایش زنده آمار ترافیک مصرفی در صفحه کاربر
- تولید QR Code
- کپی کانفیگ‌ها با یک کلیک
- جلوگیری از نام‌گذاری تکراری پروفایل‌ها در کلش

## 🚀 راهنمای نصب سریع

شما می‌توانید پنل CMH را از طریق یکی از دو روش زیر نصب کنید:

### 🤖 روش اول: نصب خودکار از طریق ربات تلگرام (پیشنهاد می‌شود)
پنل را در چند ثانیه با استفاده از ربات تلگرامی رسمی ما نصب کنید. در این روش هیچ نیازی به تنظیمات دستی و درگیری با کدهای کلودفلر نیست.

1. توکن ای‌پی‌آی کلودفلر (**Cloudflare API Token**) خود را از دشبورد کلودفلر دریافت کنید.
2. به ربات نصب‌کننده ما مراجعه کنید: [👉 برای شروع اینجا کلیک کنید 👈](https://t.me/CMH_Hub_Bot)
3. توکن API خود را برای ربات ارسال کنید و دستورالعمل‌های ساده آن را دنبال کنید. ربات، ساخت دیتابیس و نصب ورکر را به‌صورت کاملاً خودکار برایتان انجام می‌دهد.

---

### 🛠️ روش دوم: نصب دستی از طریق دشبورد کلودفلر

اگر مایلید پنل را خودتان به‌صورت دستی نصب کنید، مراحل زیر را طی کنید:

#### ۱. ساخت دیتابیس
- در دشبورد کلودفلر به مسیر **Storage & Databases** → **D1 SQLite** بروید.
- یک دیتابیس جدید (مثلاً `cf_db`) بسازید.

#### ۲. نصب ورکر و اتصال دیتابیس
- یک Cloudflare Worker جدید بسازید و کدهای پروژه را در آن قرار دهید.
- به بخش **Settings** → **Bindings** → **Add Binding** بروید.
- **مرحله بسیار مهم:** شما باید دیتابیس D1 را دقیقاً با مشخصات زیر متصل کنید:
  - نوع (Type): `D1 Database`
  - نام متغیر (Variable Name): `CMH_Hub` (باید دقیقاً همین عبارت باشد)
  - دیتابیس (Database): دیتابیسی که در مرحله قبل ساختید را انتخاب کنید.
- ورکر را Deploy (منتشر) کنید.

#### ۳. ورود به پنل مدیریت
- از طریق آدرس زیر می‌توانید به دشبورد خود دسترسی داشته باشید (به جای `<your-worker-url>` آدرس ورکر خود را قرار دهید):
  ```text
  https://<your-worker-url>.workers.dev/cmh/dash
  ```
ورود: برای اولین ورود از رمز عبور پیش‌فرض زیر استفاده کنید:
  ```text
  admin
  ```
